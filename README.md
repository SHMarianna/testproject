## Описание
Приложение для отслеживания личных расходов. Позволяет добавлять, удалять и фильтровать расходы по категории, дате и периоду.

## Функции
- Добавление расхода (сумма, категория, дата)
- Удаление выбранной записи
- Фильтрация по категории
- Фильтрация по дате
- Подсчёт суммы расходов за выбранный период
- Автоматическое сохранение в JSON
- Общая сумма всех расходов

## Проверка ввода
- Сумма — положительное число
- Дата — формат ГГГГ-ММ-ДД

python expense_tracker.py
