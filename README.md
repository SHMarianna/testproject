# Автор: Шамсудинова Марианна

## Описание
Приложение для отслеживания личных расходов. Позволяет добавлять, удалять и фильтровать расходы по категории, дате и периоду.

## Как использовать
1. Введите сумму, категорию, дату (ГГГГ-ММ-ДД) → нажмите "Добавить расход"
2. Выберите запись → "Удалить выбранный"
3. Для фильтрации: введите категорию или дату → "Применить"
4. Для подсчёта за период: укажите даты "с" и "по" → "Подсчитать сумму"
5. "Сбросить фильтры" — показать всё

## Примеры использования
- Добавление: сумма 500, категория "еда", дата 2025-04-30 → запись в таблице
- Подсчёт за период: расходы 500 и 300 в апреле, период 2025-04-01 — 2025-04-30 → сумма 800 руб
- Ошибка ввода: сумма -100 → сообщение "Сумма должна быть положительным числом"
